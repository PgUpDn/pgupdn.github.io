# pgupdn.github.io

Static personal homepage inspired by an academic profile layout and prepared for a GitHub user site.

## Files

- `index.html`: page structure and content
- `styles.css`: layout, typography, and responsive styling
- `script.js`: mobile navigation and active-section highlighting
- `assets/img/xinyu.jpeg`: profile photo

## Publish

1. Create or rename the repository to `pgupdn.github.io`.
2. Push these files to the default branch of that repository.
3. In GitHub Pages settings, make sure publishing is enabled for the default branch.
4. Visit `https://pgupdn.github.io/` after the deploy finishes.

## Included Pages files

- `.nojekyll`: tells GitHub Pages to serve the site as plain static files
- `404.html`: fallback page for missing routes

## Notes

- The site is written with plain HTML, CSS, and JavaScript, so no build step is required.
- All assets are ready to be served directly from the repository root.
- Your GitHub username can be `PgUpDn`, but the Pages repository name should still be lowercase: `pgupdn.github.io`.
